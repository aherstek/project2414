Safe School Floor Plan Dashboard

A solution centered on navigating between TCDSB schools and their associated floors, toggling on and off layers to obtain safety layers, etc.

Software used: ESRI Instant Apps

Highlights

  • Individual webmaps for each school, including floor plans with safety feature locations that can be toggled on/off
  • Access room level information (room number, student capacity, room use) by clicking on the dot located in the center of each room/li>
  • Switch between schools seamlessly by searching from list via school name or ID number
  • Additional features: change between ESRI Canada basemaps, adjust transparency of layers to view multiple at once, and save map as .pdf or screenshot to device for future reference

3D Floor Plan Render

3D Render Map

A solution designed to display the capabilities of TCDSB data through a 3D lens

Software used: ArcGIS Pro, ArcScene Viewer

Highlights

  • 3D render of school floor plans including locations of safety features and emergency exits
  • Allows for the user to gain a better understanding of school layouts, as well as room features (desks, sinks, lockers)
  • Provides TCDSB insights into 3D capabilities for future GIS initiatives

Automation Scripts / User Manual

An detailed documentation guide for TCDSB staff to seamlessly continue their indoor mapping initiatives.

Software used: Python, Microsoft Word

Highlights

  • Scripts written to assist in automating ArcGIS Pro workflow tasks such as importing .dwg files, setting coordinate systems, changing symbology, and adding joins and fields
  • Models created in ArcGIS Model Builder to further streamline workflows
  • In depth user manual for TCDSB staff to continue Safe School Floor Plan Dashboard: editing, updating, troubleshooting
  • Each workflow written in a step by step format for a non-GIS user for accessibility
